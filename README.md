# DSA Practice

A collection of Data Structures & Algorithms practice problems solved in **Java**.
Each solution is a self-contained class with a runnable `main` method, and the
commit history documents the time/space complexity for each problem.

## How to Run

Each file declares a package matching its folder name, so compile and run from the
repository root:

```bash
# Compile a single file
javac Arrays_practice/MergeSorted.java

# Run it (use the fully qualified class name)
java Arrays_practice.MergeSorted
```

Examples:

```bash
javac Hash_Table/TwoSum.java && java Hash_Table.TwoSum
javac String_Practice/AddBinary.java && java String_Practice.AddBinary
```

Sample inputs are hard-coded inside each `main` method, so no arguments are needed.
